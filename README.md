# go-ascii-animator

Yet another tool to animate your Ascii graphics

## Build

`go build main.go`

## Run

```
./main -h

Usage of ./main:
  -d float
    	Frame delay (default 0.5)
  -f string
    	File with ASCII graphics to animate
  -l	If the animation should loop
```

## ASCII files format

`[yeaa]` at the end of each block
